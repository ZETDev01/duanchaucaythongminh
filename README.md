# Chậu Cây Thông Minh V2 🌱

Hệ thống điều khiển chậu cây thông minh với Firebase và OpenWeatherMap API.

## Tính năng

- 📱 Giao diện Bootstrap responsive
- 🌿 Quản lý nhiều chậu cây
- 🌤️ Hiển thị thời tiết Hà Nội real-time
- 💧 Điều khiển bơm nước tự động
- 🔥 Tích hợp Firebase Realtime Database
- 📊 Giám sát độ ẩm đất

## Deploy lên Vercel

```bash
# Cài đặt Vercel CLI
npm install -g vercel

# Deploy
vercel
```

Hoặc kết nối GitHub repository với Vercel để tự động deploy.
